== Description ==

A Wordpress's plugin to setting up posts as test data. You can create posts or pages, indicating number of items, their titles and content.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload all files '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Finally go to the Settings -> Factory to create your posts and pages
